Steps to run the project: 

1) Install the flask:
pip install Flask

2) Enable venv: 
python -m venv venv

3) Active venv:
venv\Scripts\activate

4) Install requirements.txt.
pip install -r requirements.txt

5) To run: 
python app.py

Find the API key:
https://console.cloud.google.com/apis/library/youtube.googleapis.com
